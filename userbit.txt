Специальные биты управления  доступом
1. userbit - устанавливается только для файлов
Позволяет запустить исполняемый файл от имени владельца
Необходи для программ, выполняющихся от имени обычных пользователей, которые изменяют объекты файловой системы, защищенные от записи для всех категории пользователей. 
2. Установка userbit (в режиме администратора):
sudo chmod u+s <имя исполняемой программой>
sudo chmod u+s passwd

3. Вывод команды stat для программы passwd:
nikita@nikita-pc:~$ stat /bin/passwd
  Файл: /bin/passwd
  Размер: 93640     	Блоков: 184        Блок В/В: 4096   обычный файл
Устройство: 259/5	Инода: 1311643     Ссылки: 1
Доступ: (4755/-rwsr-xr-x)  Uid: (    0/    root)   Gid: (    0/    root)

4. В случае установки s-бита владельца для файла, у которого не юыло прав доступа на выполнение, s-бит выводиться в верхнем регистре 
stat /home/student/1.txt
Access: (4664/-rwSrw-r--)

5. Сброс user-bit (в режиме администратора):
sudo chmod u-s <имя исполняемой программы>
